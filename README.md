# React.js Grocery & Organic E-commerce Website

A complete, responsive, and premium React.js E-commerce platform designed for grocery and organic products. It features an interactive customer-facing store (`projecte`) and a dedicated dashboard (`admin-panel`) for order management.

Developed and customized by **[PODUGU MUKESH](https://github.com/mukeshpodugu)**.

---

## 🌟 Key Features

### 🛒 Customer E-commerce Store (`projecte`)
* **Interactive Shopping Cart**: Add, modify, or remove organic products seamlessly.
* **Premium Organic Vibe**: Fresh emerald green and teal responsive color scheme.
* **Firebase Integration**: Safe user authentication and real-time database capabilities.
* **Categorized Sections**: Browse through dedicated product pages for Dals, Dry Fruits, Flours, Masalas, Oils, and Rice.
* **Checkout Flow**: Complete placing order screens with billing details and email inputs.
* **Dynamic Feedback**: Real-time notifications and contact forms.

### 💼 Admin Panel Dashboard (`admin-panel`)
* **Order Tracking**: Keep track of user orders and statuses (Pending, Confirmed, Canceled).
* **Developer Control**: Easy toggles for managing active orders in the database.
* **Unified Aesthetics**: Matches the customer storefront's styling guidelines.

---

## 🛠️ Tech Stack

* **Frontend**: React.js, React Hooks, React Router DOM
* **Database & Auth**: Firebase / Firebase Firestore
* **Icons**: React Icons, FontAwesome
* **Styling**: Vanilla CSS (Premium Glassmorphism & Micro-animations)
* **Build tool**: Create React App (react-scripts)

## 🚀 Getting Started

To run the project locally, ensure you have **Node.js** and **npm** installed on your system.

### 1. Set Up the Customer Storefront (`projecte`)
```bash
cd projecte
npm install
npm start
```
The store will open at [http://localhost:3000](http://localhost:3000).

### 2. Set Up the Admin Panel (`admin-panel`)
```bash
cd ../admin-panel
npm install
npm start
```
The admin dashboard will open at [http://localhost:3000](http://localhost:3000) (or another port if 3000 is occupied).
